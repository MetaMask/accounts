# Changelog

## [1.0.0]

### Added

- Initial release of `@metamask/speculos-up`
- `ensureBinary()` for downloading and caching the Speculos binary
- `getBinaryPath()` for resolving the cached binary location
- `getPlatform()` for detecting the current OS and architecture
- SHA-256 checksum verification for downloaded binaries
- CLI entry point (`speculos-up`) for standalone binary setup
